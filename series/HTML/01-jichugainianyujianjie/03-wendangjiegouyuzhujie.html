文档结构与注解

HTML 文档结构

一个基本的 HTML 文档由以下几部分组成：

  • <!DOCTYPE html>：声明文档类型为 HTML5。
  • <html>：HTML 文档的根元素，包含整个 HTML 内容。
  • <head>：包含了文档的元数据，如标题、样式表和脚本等。
  • <title>：定义网页的标题，显示在浏览器的标签页上。
  • <body>：包含了网页的主要内容，如文本、图像、链接等。

举个例子：

<!DOCTYPE html>
<html>
<head>
  <title>我的第一个网页</title>
</head>
<body>
  <h1>欢迎来到我的网页</h1>
  <p>这是一个段落。</p>
</body>
</html>

上面这个例子展示了最基本的 HTML 文档结构，在实际开发中，一个 HTML 往往会嵌入大量标签，但是它仍然会遵守上面的基本结构。

HTML 文档注解（注释）

HTML 中的注释语法如下：

<!-- 在此处写注释 -->

它以 <!-- 开始，并以 --> 结束，被注释的内容会被 HTML 忽略。你可以用这个语法注释代码的含义，或者更直接一些，直接将代码注释掉。

除了这种最基础的注释之外，HTML 还支持 条件注释，条件注释仅仅支持判断 IE 浏览器，因此在现代开发中很少用到，但是你仍然可能在老代码中发现它们，所以你需要知道有这种东西。条件注释的语法如下：

  <!--[if IE 5]>
      <h2>版本 5</h2>
  <![endif]-->
  <!--[if IE 5.0]>
      <h2>版本 5.0</h2>
  <![endif]-->

条件注释以 <!--[if 条件]>开始，并以 <![endif]--> 结束，中间可以插入 HTML 内容，当开始部分中条件为真则中间的 HTML 会被加入到 HTML 文档，否则该注释和普通注释一样，会被 HTML 忽略。
