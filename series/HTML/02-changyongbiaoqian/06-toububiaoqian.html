头部标签

HTML <head> 元素是所有头部元素的容器。在其中可以定义一个网页的诸如标题，元信息等内容。

<title> 元素

<title> 标签定义文档的标题。它类似于一篇文章的标题，一个人的姓名，所以在 HTML 中，它是十分重要的，它的具体作用如下：

  • 定义浏览器工具栏中的标题
  • 提供页面被添加到收藏夹时显示的标题
  • 显示在搜索引擎结果中的页面标题

一个简化的 HTML 文档：

<!DOCTYPE html>
<html>
<head>
<title>Title of the document</title>
</head>

<body>
The content of the document......
</body>

</html>

<base> 元素

<base> 标签为页面上的所有链接规定默认地址或默认目标（target），即指定一个页面中所有相对URL的基准URL：

<head>
<base href="https://example.com/" />
<base target="_blank" />
</head>
<body>
    <!-- 这里的图片链接将是 https://example.com/image.png -->
    <img src="/image.png" alt="Example Image">
    <!-- 这里的链接将是 https://example.com/page -->
    <a href="page">Go to Page</a>
</body>

在 HTML 中默认的跳转行为是 _self，即当前页面，我们可以使用 base 标签的 target 属性来改变默认目标。

<meta> 标签

元数据（metadata）是关于数据的信息。

<meta> 标签提供关于 HTML 文档的元数据。元数据不会显示在页面上，但是对于机器是可读的。

<meta> 可以用于定义页面的描述信息、关键字、字符编码等，可定义范围极多，这里只对常见的类型进行介绍。

如果你需要知道更多有关的属性配置，可以参阅：(https://www.w3school.com.cn/tags/tag_meta.asp)

description 和 keywords

下面的 meta 元素定义页面的描述：

<meta name="description" content="Free Web tutorials on HTML, CSS, XML" />

下面的 meta 元素定义页面的关键词：

<meta name="keywords" content="HTML, CSS, XML" />

⚠️ description 和 keywords 已经不再作为搜索关键字

它们被设计的初期的确是为了便于搜索引擎抓取，但是由于互联网上一些人为了让自己的网页获得曝光而疯狂的增加无关的关键字，所以这一特性现在已经被弃用。

charset

charset 用于定义文档的字符编码，它可以让浏览器更容易地理解这个文档所使用的字符编码。

<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
</head>
<body>
    <!-- 页面内容 -->
</body>
</html>
