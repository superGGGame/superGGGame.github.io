表格元素

HTML 表格是用来展示数据的一种非常常见的元素。通过使用表格，可以将数据以更加直观的形式组织出来。

创建表格

表格由 <table> 标签创建，每个表格均有若干行，每行有若干列。

创建行需要使用 <tr> 标签，创建列需要使用 <td> 标签。在表格中，并不要求每一行的列数相同，你可以自由的定义，只是效果上可能就不太美观。

例子：

<table border="1">
<tr>
<td>row 1, cell 1</td>
<td>row 1, cell 2</td>
</tr>
<tr>
<td>row 2, cell 1</td>
<td>row 2, cell 2</td>
</tr>
</table>

在浏览器显示如下；

在上述案例中，需要注意的是，默认的表格并不存在边框，你可以使用 talbe 标签的 border 属性来指定边框，这会让表格看上去更清晰。

ℹ️ 不需要太在意目前的表格美观程度 或许你曾经见过在网页中出现过的一些更华丽的表格，并疑惑如何才能制作出类似的漂亮的表格。无需担心，这些是 CSS 的工作，你会在 CSS 中学习到如何去美化一个表格。 目前，你只需要掌握最基础的表格标签，它们是你创建更美观表格的基础。

表格表头

现在我们已经学会了创建一个表格，但是它还缺少一个表头来说明信息。

创建表头行的标签是：<thead>，并使用 <th> 创建表头单元格：

<table border="1">
<tr>
<th>Heading</th>
<th>Another Heading</th>
</tr>
<tr>
<td>row 1, cell 1</td>
<td>row 1, cell 2</td>
</tr>
<tr>
<td>row 2, cell 1</td>
<td>row 2, cell 2</td>
</tr>
</table>

它的效果如下：

合并单元格

有时我们可能需要合并单元格，以便更好的展示数据，我们可以使用 <td> 标签的 rowspan 和 colspan 属性来实现单元格的合并。

例子：

<table border="1">
<tr>
<th>Heading</th>
<th>Another Heading</th>
</tr>
<tr>
<td rowspan="2" >row 1, cell 1</td>
<td colspan="2" >row 1, cell 2</td>
</tr>
<tr>
<td>row 2, cell 1</td>
<td>row 2, cell 2</td>
<td>row 2, cell 3</td>
</tr>
</table>

总结

在本节中，我们学习了如何创建一个表格。

我们可以使用 <table> 标签来创建一个表格，并使用 <tr> 来定义行，使用 <td> 来定义行中的单元格。

使用 <thead> 标签来定义表头行，并使用 <th> 来创建一个表头单元。

在某些情况下会需要创建跨行或跨列的单元格，使用 <td> 的 colspan 和 rowspan 来定义这种类型的单元格。
