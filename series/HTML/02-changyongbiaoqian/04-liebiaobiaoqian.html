列表标签

在 HTML 中，列表是一种用于展示信息的常见元素。HTML 提供了三种不同类型的列表：无序列表、有序列表和描述性列表。

无序列表

无序列表使用 <ul> 元素来表示，其中每个列表项使用 <li> 元素来表示。无序列表的每个列表项通常用圆点、方块或其他符号来标记。

示例：

<ul>
  <li>苹果</li>
  <li>香蕉</li>
  <li>橙子</li>
</ul>

有序列表

有序列表使用 <ol> 元素来表示，其中每个列表项同样使用 <li> 元素来表示。有序列表的每个列表项会按照数字或字母顺序进行标记。

示例：

<ol>
  <li>第一步</li>
  <li>第二步</li>
  <li>第三步</li>
</ol>

效果如下:

描述性列表

描述性列表使用 <dl> 元素来表示，其中每个列表项由 <dt> 元素表示术语，后跟一个或多个 <dd> 元素表示解释或描述。

示例：

<dl>
  <dt>HTML</dt>
  <dd>超文本标记语言</dd>
  <dd>更多解释</dd>
  <dt>CSS</dt>
  <dd>层叠样式表</dd>
</dl>

总结

本节介绍了 HTML 中的列表标签，在 HTML 中一共有三种列表标签：

  • 有序列表
    • 使用 <ol> 创建有序列表，有序列表会用标号有序表示项目
    • 使用 <li> 创建列表项
  • 无序列表
    • 使用 <ul> 创建无序列表
    • 使用 <li> 创建列表项
  • 描述性列表
    • 使用 <dl> 创建描述性列表
    • 使用 <dt> 创建术语项
    • 使用 <dd> 创建解释项
