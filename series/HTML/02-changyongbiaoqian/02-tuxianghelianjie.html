图像和链接

HTML 链接

链接是网页开发中非常重要的一部分，我们可以使用 <a> 标签创建。

创建链接

要创建一个链接，需要在 <a> 标签中使用 href 属性来指定要链接到的目标地址。例如，你要创建一个指向百度的链接：

<a href="https://www.baidu.com">点击这里</a>

这样你就创建了一个指向百度的超链接。

链接向本地文件

如果在你创建的 html 页面文件同一文件夹中，存在另一个页面，那你也可以使用链接指向它，创建一个链接。

<a href="about.html">关于我们</a>

上面的例子中创建的链接，会指向和该文件相同目录下的 about.html 文件。

链接到页面内部

<a> 标签可以指向某一个页面的内部的某一部分，这个称作锚点。

所谓的锚点就是在元素上使用 id 属性给与一个字符串，例如：

<a id="section1"></a>
<h2>第一节</h2>
<p>这是第一节的内容。</p>

然后在页面的其他位置创建一个链接，指向这个锚点：

<a href="#section1">跳转到第一节</a>

当你点击这个链接的时候，页面会定位到上面的 “锚点” 。

链接到电子邮件地址

如果你想在页面上提供一个链接，让用户可以发送电子邮件给你，可以使用 mailto 协议。比如：

<a href="mailto:youremail@example.com">联系我们</a>

用户点击这个链接后，会自动打开默认的邮件客户端，并填写你指定的电子邮件地址。

以上是 html 的基本用法，通过合理的利用这些可以让你的网页浏览更加流畅。

HTML 图像

在网页开发中，图像是非常重要的元素之一，能够丰富页面内容，提升用户体验。在 HTML 中，我们可以通过 <img> 标签来插入图像。

创建图像元素

创建图像元素，使用 <img> 标签，并在其 src 属性中指定图像的 URL。例如：

<img src="https://example.com/image.jpg" alt="描述图像的文字">

alt 属性用于提供图像的文本描述，这对于视觉障碍用户和无法加载图像的情况非常重要。

图像大小

如果需要控制图像的大小，可以使用 width 和 height 属性。例如：

<img src="https://example.com/image.jpg" alt="描述图像的文字" width="200" height="150">

在上面的示例中，width 和 height 属性分别指定了图像的宽度和高度，以像素为单位。

图像链接

图像标签也可以结合链接标签使用，只需要将 <img> 标签包裹在 <a> 标签内即可。例如：

<a href="https://example.com">
    <img src="https://example.com/image.jpg" alt="描述图像的文字">
</a>

在上面的示例中，点击图像将会跳转到指定的链接地址。

总结

在本节内容，我们学习了：

  • HTML 链接
    • 使用<a>标签创建一个链接。
    • 使用标签的 href 属性来指定链接地址，它可以是一个网页地址，可以是一个锚点，甚至是一个电子邮件地址。
  • HTML 图像
    • 创建图像元素使用 <img> 标签
    • 使用标签的 src 属性来指定资源路径。
    • 可以使用 width 和 height 属性来控制图片大小
    • 可以结合 <a> 标签来创建图片链接，创建的方式是将 <img> 标签包裹在 <a> 标签中。
