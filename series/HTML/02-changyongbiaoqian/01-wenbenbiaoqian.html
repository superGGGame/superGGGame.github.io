文本标签

HTML 标题

标题（Heading）是通过 <h1> - <h6> 等标签进行定义的。

<h1> 是最大的标签，<h6> 是最小的标签，它们很简单。

它们都是块级元素。

<h1>This is a heading</h1>
<h2>This is a heading</h2>
<h3>This is a heading</h3>

HTML 段落

段落是通过 <p> 标签定义的，它是一个块级元素。

<p>This is a paragraph</p>
<p>This is another paragraph</p>

加粗标签：b 和 strong

标签加粗有两种标签形式，分别是 b 和 strong。它们是行内元素。

<b>我被加粗了</b>
<strong>我被加粗了</strong>

它们的区别在于， <string> 是语义化标签，很明显可以发现，它的标签名称是一个单词，而 <b> 只是一个字母。

ℹ️ 语义化标签的作用 语义化标签除了更加清晰地表明了标签的作用之外，还能为浏览器解析文档提供帮助。比如 <strong> 标签的内容在盲人模式下会被读两遍，而 <b> 标签并不会被重读。

斜体标签 em

斜体标签顾名思义，可以让内容斜体展示。它是一个行内元素。

<em>我是一段斜体文字</em>

换行标签

在 HTML 中，直接换行是无法被正确解析的，在源代码中换行会被认为是一个代码分割，并不会提现在页面效果上。

在 HTML 中，如果你希望创建换行，请使用 <br> 标签来创建。

<br>

tip: 不要使用类似空内容的 <p></p> 标签来代替 <br> ，这不是一个好习惯。

⚠️ 需要注意，虽然 <br> 会有换行的效果，这让他看起来似乎有独占一行的效果，但它实际上是一个行内元素。

字符实体

在 HTML 中，有些特殊字符是无法直接显示的，它们被预留了出来。比如在 HTML 中无法直接使用大于号和小于号，因为浏览器会认为这是标签的开始或结束。

字符实体类似这样：

&entity_name;
<!-- 或者 -->
&#entity_number;

例如显示小于号，你需要使用 &lt; 或 &#60;。

建议使用实体名，因为它们更容易被识别和记忆。

⚠️ 注意，在 HTML 中连续的空格不管是多少个都会被缩减为一个，如果你需要插入多个空格，需要使用字符实体 &nbsp; ，这是相当常用的一个字符实体。

总结

本节介绍了几种常见的文本标签：

  • 标题标签：使用 <h1> ~ <h6> 来创建标题，数字越小对应的标签越小，它们是块级元素。
  • 段落标签：使用 <p> 标签创建段落，它是块级元素。
  • 加粗标签：
    • <b>： 加粗标签，行内元素
    • <strong>：语义化加粗标签，行内元素
  • 斜体标签：使用 em 创建斜体文字，行内元素。
  • 换行标签：<br> 创建换行，行内元素。

认识了字符实体的概念，在 HTML 中有部分字符被预留了出来，要使用它们需要使用字符实体，它们的格式如下：

&entity_name;
<!-- 或者 -->
&#entity_number;

空格是常用的字符实体，它的实体名称是： &nbsp;

文本标签有非常多种，但是你没有必要记住所有的标签，只需要掌握常用的标签即可。对于更多的标签，你可以参照 MDNopen in new window。
